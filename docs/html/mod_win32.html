win32 모듈¶

win32 모듈이 내장하고 있는 함수는 UI 클래스의 기능을 대부분 탑재하고 있습니다.

autowinpy.win32.get_windows() → List[int]¶

핸들 리스트 출력.

반환값

[int] 윈도우 핸들

예제

import autowinpy as awp
from win32 import win32gui

window_list = awp.win32.get_windows()
for hwnd in window_list:
    print(win32gui.GetWindowText(hwnd))

현재 윈도우 핸들을 나열합니다.

autowinpy.win32.get_child_windows(hwnd: int) → List[int]¶

자식 핸들 리스트 출력.

매개변수

hwnd (HWND) – 윈도우 핸들

반환값

([HWND]) 자식 윈도우/컨트롤의 핸들 리스트

예제

import autowinpy as awp
from win32 import win32gui

window_list = get_windows()
for hwnd in window_list:
    print(win32gui.GetWindowText(hwnd))
    childs_list = get_child_windows(hwnd)
    for chwnd in childs_list:
        print("\t", win32gui.GetWindowText(chwnd))

현재 윈도우와 자식 핸들 명칭을 나열합니다.

autowinpy.win32.get_window_rect(toHWND: int, fromHWND: int = None) → Tuple[int, int, int, int]¶

윈도우가 그리는 사각영역의 위치좌표를 얻습니다.

매개변수

  • toHWND (HWND) – 영역좌표를 알고 싶은 윈도우의 핸들.

  • fromHWND (HWND) – (옵션) 상대위치를 구하고자 할 때 기준점이 될 윈도우

반환값

(left, top, right, bottom)

계산 후 출력하는 좌표는 모니터의 우측 상단이 원점이며, 오른쪽으로 갈수록 x값이 증가하고, 아래로 내려갈수록 y값이 증가합니다.

상대좌표는 기준점이 되는 창의 왼쪽 끝을 새로운 원점으로 하며, 오른쪽 아래로 갈수록 값이 증가합니다.

autowinpy.win32.get_window_view_array(hwnd: int) → numpy.ndarray¶

윈도우 화면을 가져옵니다.

매개변수

hwnd (HWND) – 윈도우 핸들

반환값

(opencv.mat) 선택한 창의 내용을 OpenCV나 Numpy에서 사용할 수 있는 BGR이미지를 출력합니다.
