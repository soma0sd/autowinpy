UI 클래스¶

class autowinpy.UI(hwnd, parent: autowinpy.UI = None)¶

기반 클래스: object

윈도우와 컨트롤을 핸들링하기 위한 클래스.

매개변수

  • hwnd (int) – 윈도우 핸들.

  • [옵션]parent (UI) – 부모 (윈도우)

UI클래스 는 윈도우가 비활성 상태일 때도 캡쳐나 클릭 등을 수행하기 위한 클래스입니다. win32.get_windows() 는 현재 실행중인 윈도우와 컨트롤로부터 UI 클래스 요소를 만드는 손쉬운 방법을 제공합니다.

parent 매개변수는 UI가 특정 윈도우의 컨트롤인 경우에 사용합니다. 윈도우에서 컨트롤의 상대적인 위치 등을 계산하기 위해 필요합니다.

property childs¶

자식 목록.

반환값

([UI]) 자식 윈도우/컨트롤 목록.

property hwnd¶

핸들 출력.

반환값

(int) 윈도우의 고유 핸들

property is_active¶

제어 가능 여부.

반환값

(bool) 패키지로 제어할 수 있는 요소인지 확인합니다.

property is_child¶

부모 존재 확인.

반환값

(bool) 부모가 있는 UI인지 확인합니다.

property name¶

이름.

반환값

(str) 현재 윈도우가 가지고 있는 문자열

property parent¶

부모 UI.

반환값

(UI) 부모가 있는 경우 부모 UI를 반환.

(None) 부모가 없는 경우.

property rect¶

윈도우 영역.

반환값

(left, top, right, bottom) 윈도우 영역의 좌표. 부모가 있는 경우에는 부모를 기준으로 하는 상대좌표.

자세한 내용은 win32.get_window_rect() 참조.

property screen_array¶

스크린 캡쳐.

반환값

(opencv.mat)윈도우 스크린샷 이미지.

자세한 내용은 win32.get_window_view_array() 참조.

property size¶

윈도우 크기.

반환값

(width, height) 윈도우의 폭과 높이.
