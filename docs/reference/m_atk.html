autowinpy.atk.image_tk(image: autowinpy.type.Image, dsize: Tuple[int, int] = None, fill: bool = False) → ImageTk¶

autowinpy의 이미지를 tkinter 이미지로 변환

매개변수

  • image – 변환할 이미지

  • dsize – [옵션] 기본값 None

  • fill – [옵션] 기본값 False

반환값

tkinter 이미지

class autowinpy.atk.GuiSelectCombo(master=None, window_combo: Optional[autowinpy.atk.GuiSelectCombo] = None, **kw)¶

tkinter를 위한 Gui 선택 콤보박스

이 클래스는 tkinter.ttk.Combobox 를 상속합니다.

매개변수

  • master – 상위 Tkinter 요소

  • window_combo – 상위 GuiSelectCombo 를 매개변수로 입력하면, 이 클래스는 자식 선택 콤보박스로 동작합니다.

  • **kw – Tkinter 위젯 설정
